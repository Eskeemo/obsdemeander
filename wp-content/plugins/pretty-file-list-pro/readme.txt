The File List Pro plugin lists files attached to a post, displays them, and pages them in an attractive format without you having to do any work. It iss designed to be very easy to use, and even comes with built in styles for different set ups.

== Description ==

The File List Pro plugin lists files attached to the current post of page. All you need to do is add the shortcode to your page or post, specify the types of files you want displayed, and pick the number of files you want per page.
